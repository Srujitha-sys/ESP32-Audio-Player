# ESP32 Audio Player

An IoT-based Audio Player using ESP32 that allows users to control audio playback through a web interface.

## Features
- Play and Pause Audio
- Volume Control
- Bass Control
- Web-Based Interface
- Wi-Fi Connectivity
- DAC Audio Output

## Hardware Required
- ESP32 Development Board
- Speaker
- Audio Amplifier (Optional)
- Resistors and Capacitors (Optional)

## Connections

| ESP32 Pin | Component |
|------------|-----------|
| GPIO25 | Speaker/Amplifier Input |
| GND | Speaker GND |
| 3.3V/5V | Power Supply |

## Technologies Used
- ESP32
- Arduino IDE
- WiFi.h
- WebServer.h
